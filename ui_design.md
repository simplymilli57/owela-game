Owela App User Interface Design

## Opening Screen (Home)

The first screen shows:

- Owela logo
- Play Game button
- Learn Rules button
- Statistics button
- Settings button

## Game Screen

The game screen contains:

- 60-hole board
- Seeds inside holes
- Player 1 and Player 2 indicators
- Current player turn
- Captured seeds counter
- Pause button

## Rules Screen

Shows:

- How to play Owela
- How seeds move
- How capturing works
- How to win

## Settings Screen

Options:

- Language selection
- Sound on/off
- Game preferences

## Design Style

The app should feel traditional and cultural:

- Wooden board appearance
- Realistic seeds
- Smooth animations
- Simple controls
- Easy for all ages to use
