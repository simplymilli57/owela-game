Owela Game Rules

A digital version of the traditional Owela game for iPhone and Android.

## Game Setup
- 60 holes total
- 2 players
- Each player controls 30 holes
- Players face each other
- Movement direction: counter-clockwise

## Starting Position
- Each player has 19 active holes
- Each filled hole starts with 2 seeds
- 15 filled holes in one row
- 4 filled holes in the inner row

## Rules
- Players choose a hole on their side.
- Seeds are distributed one by one.
- If the last seed lands in an occupied hole, sowing continues.
- A turn ends when the last seed lands in an empty hole.

## Capturing
- Captures happen when the final seed lands in an occupied inner-row hole.
- Opposite opponent seeds can be captured.

## Goal
The player with the most captured seeds wins.
