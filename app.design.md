# Owela App Design

## Main Menu
When the app opens, players see:

- Play Owela
- Learn the Rules
- Statistics
- Settings

## Game Screen

The game screen will show:

- 60-hole Owela board
- Player 1 seeds
- Player 2 seeds
- Turn indicator
- Capture score
- Restart button

## Game Modes

Version 1:
- Two players on one device

Future versions:
- Player vs Computer
- Online multiplayer

## Settings

Options:
- Language selection
- Sound effects
- Rules explanation

## Goal

Create a digital Owela experience that feels like the traditional board game.
